# Day 3 – Linux & Log Analysis

## Overview  
This repository contains my Day 3 learning focused on Linux basics and log analysis from a SOC (Security Operations Center) perspective.

---

## Topics Covered  
- Basic Linux commands (ls, pwd, cd, cat)  
- Directory navigation (/var/log)  
- File creation and handling  
- System log analysis (syslog)  
- Authentication log analysis (auth.log)  
- Keyword search using grep  

---

## Key Learning  
- Linux commands are essential for system navigation and analysis  
- Logs help monitor system activities and detect threats  
- Failed login attempts may indicate unauthorized access or brute force attacks  
- Log analysis is a core skill for SOC analysts  

---

## Commands Practiced  
```bash
ls
pwd
cd /var/log
cat /var/log/syslog
cat /var/log/auth.log
grep "Failed" /var/log/auth.log
```

---

## Conclusion  
This session helped me understand how to work with Linux and analyze logs to identify potential security threats in real-world scenarios.
